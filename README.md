site_estudo_front-end

Este repositório contém arquivos HTML utilizados para estudo de desenvolvimento Front-End. O objetivo é praticar a criação de páginas utilizando os fundamentos de HTML e desenvolver uma melhor compreensão da estrutura de um site básico.

Conteúdo do repositório

O projeto está organizado com os seguintes arquivos:

bg_atalho.html – Exemplo de texto de estudo para prática de estrutura HTML.

site_exercicio.html – Página com seções sobre conceitos introdutórios de Front-End como HTML, CSS e JavaScript.

bg-3.jpg – Imagem de fundo ou recurso visual utilizado no projeto.

Sobre o projeto

O foco desse repositório é servir como um ambiente de prática para quem está começando no desenvolvimento web, reforçando o uso de:

Estrutura básica de documentos HTML

Elementos de texto e hierarquia de cabeçalhos

Organização de seções em uma página

Esses exemplos podem ser abertos diretamente em um navegador para visualização e estudo.

Como abrir

Clone ou faça download deste repositório.

Abra um dos arquivos .html no navegador de sua preferência.

Analise o código e pratique modificações para aprender como cada elemento HTML funciona.

Tecnologias utilizadas

Este projeto foi implementado utilizando apenas:

HTML
